融资过程中，创业者必须对答如流的50个问题

2017-02-12

      很多创业者在没有做好充分准备的情况下就贸然开始融资了，这对他们显然是非常不利的。在融资过程中，投资人一般会问创业者什么问题呢？以下的50个问题可供大家参考：

      团队：

      1. 是什么启发了你，让你决定全职创业？

      2. 你团队里的人各自负责什么？

      3. 你们的经历对这次创业有什么帮助？

      4. 你接下来准备招哪些人？为什么？

      5. 你关系最好的导师/顾问是谁？

      6. 你最喜欢哪本书？

      市场：

      1. 你要解决的问题是什么？

      2. 这个问题的痛点有多痛？

      3. 你的目标用户是谁？

      4. 市场有多大？

      5. 让我再问一遍，市场到底有多大？

      6. 你能给我发几篇文章，帮助我了解这个市场吗？

      产品：

      1. 为什么你的点子可以独立成为一个项目而不是某个现有产品的一项功能？

      2. 如果你现在可以为你的产品增加两项新功能，它们会是什么？

      3. 你的产品对技术的要求有多高？

      4. 你的产品使用起来方便吗？

      5. 你产品/模式的壁垒是什么？

      流量：

      1. 你目前有多少用户？

      2. 你的用户有多活跃？

      3. 公司目前实现了哪些里程碑？

      4. 公司上个月的出色业绩是什么？

      5. 你至今的增长量来自哪里？

      6. 你从你的早期用户身上学到的最重要的东西是什么？

      7. 你能分享一些用户评价吗？

      商业模式：

      1. 你如何获客？

      2. 你的销售周期有多长？

      3. 你的单位经济效益是多少？它是否会随着公司规模的扩大而提高？

      4. 你准备怎样实现盈利？

      5. 公司目前发展路线的潜在风险是什么？

      6. 你能跟我梳理一下你的供应链吗？它会随着公司规模的扩大而获得怎样的优化？

      竞争：

      1. 谁是你最直接的竞争对手？

      2. 和竞争对手相比，你的独特性（定价、产品等）是什么？

      3. 为什么这个痛点需要一家创业公司而不是一家大公司来解决？

      4. 有其他人试图解决过这个痛点吗？为什么他们失败了？为什么你可以？

      5. 你最尊敬的公司是哪一家？

      规模：

      1. 随着公司规模变大，是否会带来网络效应？

      2. 你的目标用户定位在普罗大众还是小众人群？为什么？

      3. 你的公司准备上市吗？

      4. 公司需要达到什么阶段你才会说“我们做到了自己想做的事”？

      5. 达到那个阶段之后，你准备做什么？

      监管合规：

      1. 你面临哪些监管障碍？

      2. 你是否申请过专利？

      3. 你如何为自己的产品创造壁垒？

      4. 公司最大的监管风险是什么？

      融资：

      1. 你为什么选择现在融资？

      2. 你至今的流量证明了什么？

      3. 你融资是为了什么？

      4. 这一轮融资的时间你是如何计划的？

      5. 你参考了哪些公司的估值？

      6. 你希望哪3个天使投资人或者哪5家VC机构参与？
